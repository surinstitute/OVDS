# Group Schema

Schema file: [spec/v0.2.0/group.schema.json](../../spec/v0.2.0/group.schema.json)

## Purpose

The group schema defines a reusable automotive group entity above makes.

## Current Scope

The v0.1 schema currently captures:

- group identifier
- display name

Examples include multi-brand parent organizations such as Stellantis.
