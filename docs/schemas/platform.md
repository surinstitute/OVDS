# Platform Schema

Schema file: [spec/v0.2.0/platform.schema.json](../../spec/v0.2.0/platform.schema.json)

## Purpose

The platform schema defines a reusable vehicle platform entity that can be shared across makes and groups.

## Current Scope

The v0.1 schema currently captures:

- platform identifier
- display name
- associated group identifiers
- associated make identifiers

This allows OVDS to represent cross-brand architectures such as shared modular platforms.
