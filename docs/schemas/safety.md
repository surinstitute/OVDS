# Safety Schema

Schema file: [spec/v0.1.1/safety.schema.json](../../spec/v0.1.1/safety.schema.json)

## Purpose

The safety schema defines normalized, comparable safety features independent of commercial package naming.

## Current Scope

The v0.1 schema groups boolean safety features under stable categories:

- collision warnings
- collision intervention
- driving control assistance
- rear seat safety
- visibility and control
- restraints

This first version prioritizes cross-brand comparability over commercial packaging detail.

## Evolution

The taxonomy is expected to evolve in future versions as feature distinctions become more precise. The current grouped structure is intended to leave that path open without changing the role of the schema.
