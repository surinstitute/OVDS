# Engine Schema

Schema file: [spec/v0.1/engine.schema.json](../../spec/v0.1/engine.schema.json)

## Purpose

The engine schema defines a reusable internal combustion engine entity.

## Current Scope

The v0.1 schema currently captures a minimal but reusable engine record:

- engine identifier
- primary fuel type
- displacement
- cylinder count
- aspiration
- layout

This schema is intended to be referenced by vehicles, generations, variants, or future fitment/application records.
