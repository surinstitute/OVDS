# Make Schema

Schema file: [spec/v0.1/make.schema.json](../../spec/v0.1/make.schema.json)

## Purpose

The make schema defines a reusable brand or marque entity.

## Current Scope

The v0.1 schema currently captures:

- make identifier
- associated group identifiers
- display name

This allows a make to be modeled independently from any single vehicle record and linked upward to one or more automotive groups.
