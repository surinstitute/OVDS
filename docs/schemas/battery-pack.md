# Battery Pack Schema

Schema file: [spec/v0.1.2/battery-pack.schema.json](../../spec/v0.1.2/battery-pack.schema.json)

## Purpose

The battery pack schema defines a reusable high-voltage battery pack entity.

## Current Scope

The v0.1 schema currently captures:

- battery pack identifier
- cell chemistry
- gross capacity
- usable capacity
- nominal system voltage

This entity is intended for reuse across hybrid, plug-in hybrid, battery electric, and fuel-cell vehicle records where applicable.
